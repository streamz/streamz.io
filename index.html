streamz.io

big memory stream processing

Download this project as a .zip file Download this project as a tar.gz file

streamz is a distributed realtime compute framework. It is designed to process streaming event sources (such as feeds, server logs etc...) on a multi-tenant compute cluster. The programming paradigm is similar to map-reduce, scatter/gather and CEP (complex event processing). streamz is simple. No configuration files, just a script or a few lines of code and streamz is processing "big data" at scale.
